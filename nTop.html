Project Overview

This project was built to improve structural performance by combining FEM simulation, automated result extraction, and iterative design updates.

The workflow reads simulation outputs such as stress and strain energy density, evaluates local behavior across the structure, and adjusts geometry parameters accordingly.

The target was to move toward a more uniform energy distribution while also controlling deformation, mass, and stress concentration. The process was developed to be practical, repeatable, and suitable for engineering optimization studies.

void EvaluateResults_SED(int count, float[] prev)
{
    float t_old;
    int counter = 0;

    for (int i = 0; i < lc_W; i++)
    {
        for (int j = 0; j < lc_V; j++)
        {
            for (int k = 0; k < lc_U; k++)
            {
                var tile = TileStructure[i, j, k];

                for (int l = L_def; l < L_def + L_min; l++)
                {
                    t_old = prev[counter];

                    if (tile.SubTiles[l].SED < (SED_Val / SafetyFactorSED))
                        tile.SubTiles[l].Thickness = t_old * 0.8;
                    else
                        tile.SubTiles[l].Thickness = t_old * 1.2;

                    counter++;
                }
            }
        }
    }
}
