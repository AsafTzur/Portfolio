Project Overview

This project involved building a complete Finite Element Method (FEM) solver from first principles, without relying on external libraries. The goal was to gain full control over the numerical formulation, implementation, and validation of the method.

The solver supports multiple element types including linear triangles (T3), quadratic triangles (T6), and biquadratic quadrilaterals (Q9). Each element is implemented using isoparametric formulation, with shape functions, Jacobian transformation, and Gauss integration handled explicitly.

The system assembles global stiffness matrices, applies boundary conditions, and solves PDEs of the form typically encountered in structural and membrane problems.

Verification was performed using benchmark problems with known analytical solutions, including Poisson equations and circular membrane cases. Convergence behavior was evaluated using L2 and H1 norms, providing insight into element performance and numerical accuracy.

The project highlights a deep understanding of FEM theory, numerical implementation, and practical challenges such as mesh quality, element order behavior, and integration accuracy.

Code Snippet

function [Ke] = localStiffness(elemNodes, dNdx, dNdy, w, detJ)

Ke = zeros(length(elemNodes));

for i = 1:length(w)
    B = [dNdx(:,i)'; dNdy(:,i)'];
    Ke = Ke + (B' * B) * detJ(i) * w(i);
end

end
